The /w command is a Claude Code tool that solves session context recovery—finding previous work you've done but can't remember which session it was in.

Usage: `/w that one thing that one time` — searches across your transcripts, sessions, and git history to locate and resume previous work context.

## Key Features

- **Cross-Source Search**: Scans transcripts, Claude Code sessions, and git history
- **Natural Language Queries**: Describe what you worked on in plain language
- **Session Resumption**: Quickly jump back into previous work context
- **Git Integration**: Searches commit history and branch context
- **Transcript Search**: Finds relevant conversation history

## Use Cases

- **Context Recovery**: Find previous work you vaguely remember but can't locate
- **Session Continuation**: Resume interrupted projects with full context
- **Reference Lookup**: Search for similar problems you've solved before
- **Workflow Optimization**: Reduce setup time when switching between projects
