Nate Berkopec's dotfiles repo, bookmarked specifically for its Claude Code skills implementation. The repo includes a `readme-writer` skill under `.claude/skills/` that demonstrates skills as more than plain markdown — incorporating scripts and tooling to give Claude actual capabilities rather than just context. Berkopec argues this is the right approach: skills should provide tools, not just prompts.

## Key Features

- Claude Code skills with embedded scripts (not just .md files)
- readme-writer skill as a concrete example of tool-providing skills
- Mac dotfiles setup: Homebrew, SSH keys, Fish shell, macOS defaults
- Ruby-based step system for idempotent environment setup
- Ubuntu 22.04 support with GUI test container via noVNC
