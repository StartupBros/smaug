Anthropic's official Skill Creator skill for Claude Code, which includes a built-in eval framework that spawns child agents to test generated skills and iteratively drive improvement. The skill encodes years of researcher experience in prompt/skill design and produces surprisingly effective results. Part of the broader `anthropics/skills` repository (105k+ stars) that contains the canonical Agent Skills specification and a library of example skills for creative, technical, enterprise, and document tasks.

## Key Features

- Built-in eval framework that runs tests using child agents to assess skill effectiveness
- Iterative improvement loop driven by automated test results
- Encodes Anthropic researcher expertise in skill engineering patterns
- Part of a public repository with Apache 2.0 licensed example skills
- Also ships source-available document skills (DOCX, PDF, PPTX, XLSX) used in production Claude.ai
