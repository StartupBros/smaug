Goose is an open-source, extensible local AI agent built by Block (Square/Cash App) in Rust. It goes beyond code suggestions to autonomously install, execute, edit, and test — building entire projects, debugging failures, orchestrating workflows, and interacting with external APIs. Works with any LLM and supports multi-model configuration for cost/performance optimization. Available as both desktop app and CLI.

Jack Dorsey specifically called out the under-the-hood quality, noting the interface still needs work but the team is pushing improvements. With 34,000+ stars, it's one of the more prominent open-source agent projects. MCP integration is a first-class feature.

## Key Features

- Works with any LLM; multi-model configuration supported
- MCP server integration built-in
- Available as both desktop app and CLI
- Builds full projects autonomously: write, execute, debug, orchestrate
- Custom distributions supported — build your own goose distro with preconfigured providers, extensions, and branding
- Written in Rust for performance
