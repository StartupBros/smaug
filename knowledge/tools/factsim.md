FactSim is an AI-powered life trajectory simulator that learns about a user and models their behavior using agents. Users can test different life paths, run scenarios, and explore outcomes across health, relationships, money, and career — described as "your life in sandbox mode."

## Key Features

- **Behavioral modeling**: Learns individual user patterns to make simulations personalized rather than generic
- **Multi-domain simulation**: Models health, relationships, money, career, and more in parallel
- **Scenario testing**: Lets users explore "what if" paths before committing to them
- **Agent-driven**: Uses AI agents to model behavior and simulate trajectory outcomes
