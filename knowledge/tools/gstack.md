gstack is a collection of 15 specialist Claude Code skills built by Garry Tan (CEO of Y Combinator) that gives AI agents clearly defined engineering roles — CEO, CTO, designer, QA engineer, release manager — so they know exactly what to do. Designed to be installed as a Claude skills directory and run simultaneously across multiple agents.

## Key Features

- 15 specialist skill roles covering planning, building, shipping, and safety
- `/office-hours` — turns rough ideas into real plans with three implementation approaches
- `/plan-ceo-review` — challenges scope like a CEO, finds the 10-star product in your feature list
- `/plan-eng-review` — locks down architecture with diagrams, edge cases, and failure modes
- `/review` — code review that auto-fixes obvious bugs
- `/qa` — opens a real browser, clicks through the app, finds and fixes bugs, writes regression tests
- `/ship` — syncs code, runs tests, checks coverage, pushes to GitHub, opens PR in one command
- `/document-release` — updates README, architecture, and contributing docs after each ship
- `/careful` — warns before any destructive operation
- `/freeze` — locks all files except the active working folder
- Run 10–15 skills simultaneously across parallel agents
