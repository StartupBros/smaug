companies.sh is an open standard and directory for "Agent Companies" — pre-packaged multi-agent systems that can be imported and run with a single CLI command. The concept positions itself as an App Store moment for agentic workflows: instead of configuring individual agents, you install an entire company's worth of coordinated agents at once.

The CLI uses `npx companies.sh add <repo/company>` to pull a company package from the directory, making it trivially easy to share and distribute complex agent configurations.

## Key Features

- Open standard for packaging and distributing multi-agent "companies"
- Single-command install via `npx companies.sh add <repo/company>`
- Public directory at companies.sh for discovering available agent companies
- Designed for sharing and reuse of complex agentic workflows
