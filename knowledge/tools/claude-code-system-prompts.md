A continuously-updated repository that extracts and publishes all of Claude Code's system prompts, tool descriptions, sub-agent prompts, and utility prompts directly from the compiled npm package source. Maintained by Piebald AI, it tracks 110+ prompt strings across every Claude Code release version with a detailed changelog. The repo also surfaces Claude Code's ~40 system reminders, making it the most comprehensive public reference for understanding how Claude Code is instructed to behave.

## Key Features

- Full extraction of all Claude Code system prompts, updated within minutes of each release
- Includes prompts for builtin tools (Write, Bash, TodoWrite, etc.), sub-agents (Plan, Explore, Task), and utility functions (CLAUDE.md generation, compaction, etc.)
- CHANGELOG.md tracks prompt evolution across 132+ versions since v2.0.14
- Companion tool `tweakcc` allows patching your local Claude Code installation with customized prompt variants
