A curated collection of agent skills for AI coding assistants, following the [Agent Skills Standard](https://github.com/anthropics/agent-skills-standard). Compatible with Claude Code, Cursor, Gemini Code Assist, GitHub Copilot, and other tools.

## Key Features

- Install via `npx skills add Shpigford/skills` or manual git clone
- Each skill has trigger phrases that auto-activate it in compatible tools
- Context: shared in conversation where @Shpigford mentions turning a repeated "build a chat widget" task into a reusable skill

## Available Skills

| Skill | Description |
|-------|-------------|
| `build` | Feature development pipeline — research, plan, track, implement (4-phase) |
| `but-for-real` | Skeptical second pass on own work — catches wrong logic, forgotten tests, scope creep |
| `chat-widget` | Build real-time support chat with floating widget + admin dashboard |
| `conductor-setup` | Configure Rails projects for Conductor (Mac parallel coding agents app) |
| `favicon` | Generate complete favicon sets from source images (requires ImageMagick v7+) |
| `feature-image` | Generate branded social media images for feature announcements |
| `issues` | Create, list, and view GitHub issues from chat |
| `new-rails-project` | Create opinionated Rails 8 + React projects |
| `readme` | Generate comprehensive project documentation with architecture overview |
| `screenshots` | Generate marketing screenshots with Playwright |
