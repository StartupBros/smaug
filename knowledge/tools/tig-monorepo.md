# TIG Monorepo (tig-foundation/tig-monorepo)

## Summary

The TIG (Trusted Innovation Graph) Foundation is a proof-of-work blockchain where each block challenge is an NP-hard optimisation problem. Solvers must find valid algorithm submissions that beat a benchmark threshold — the work is hard to find but cheap to verify — creating a cryptoeconomic incentive structure that coordinates an untrusted pool of distributed compute contributors toward real algorithmic innovation.

## Key Concepts

- **Computational asymmetry:** challenges are NP-hard to solve but polynomial to verify, mirroring proof-of-work
- **Algorithm commits:** submissions build on each other in a DAG structure, rewarding genuine improvement
- **Untrusted distributed pool:** anyone can contribute compute; the blockchain enforces correctness without requiring trust
- **Challenge domains:** includes combinatorial optimisation problems (e.g. c006 satisfiability variants at play.tig.foundation)

## Relevance

Proposed as a candidate architecture for decentralised AI training coordination — described by founder John Fletcher (PhD, Cambridge) as matching the ideas Karpathy discussed on the No Priors podcast regarding commits that build on each other with cryptoeconomic verification.
