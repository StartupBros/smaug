Paperclip is an open-source orchestration platform for autonomous AI companies. It provides a Node.js server and React UI that coordinates a team of AI agents to run a business, with built-in org charts, budgets, governance, goal alignment, and agent coordination features.

If OpenClaw is an employee, Paperclip is the company. Instead of managing pull requests, you manage business goals by defining objectives, assembling a team of agents (from any provider), approving their strategy, and monitoring progress from a unified dashboard.

## Key Features

- Bring your own agents (OpenClaw, Claude, Codex, Cursor, Bash, HTTP, or custom)
- Goal alignment - every task traces back to company mission
- Agent heartbeats for autonomous operation on scheduled intervals
- Cost control with per-agent monthly budgets
- Multi-company support with complete data isolation
- Full ticket system with tool-call tracing and audit logs
- Governance controls (approve hires, override strategy, pause/terminate agents)
- Org charts with hierarchies, roles, and reporting lines
- 24/7 autonomous operation with human oversight

## Quick Start

```bash
npx paperclipai onboard
```
