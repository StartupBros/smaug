Ghidra is a software reverse engineering (SRE) framework created and maintained by the NSA Research Directorate. It provides a full suite of high-end software analysis tools for analyzing compiled code across Windows, macOS, and Linux. Supports disassembly, assembly, decompilation, graphing, and scripting. Can be extended with custom plugins in Java or Python, and supports a wide variety of processor instruction sets and executable formats. Bookmarked in the context of testing AI-assisted reverse engineering with GLM-5.1 and the Hermes framework.

## Key Features

- Disassembly, decompilation, and assembly for a wide range of CPU architectures and binary formats
- Interactive GUI for code analysis as well as headless/automated mode for scripting
- Extensible via Java or Python scripts and Eclipse/VS Code plugin integrations
- Collaborative SRE support for teams tackling complex binaries
- Open source (Apache 2.0) with 67k+ GitHub stars, originally built to solve NSA-scale analysis problems
