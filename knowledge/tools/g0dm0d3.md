G0DM0D3 is a single-file (index.html), open-source, privacy-respecting chat interface that queries 50+ AI models in parallel via OpenRouter and uses a scoring agent ("the Tastemaker") to surface the highest-quality response. Built by red-teaming researcher Pliny the Liberator, it is designed for studying model robustness and the limits of post-training alignment layers.

## Key Features

- **GODMODE CLASSIC** — 5 battle-tested model + prompt combos race in parallel; best response wins
- **ULTRAPLINIAN** — Multi-tier evaluation engine querying 10–51 models, composite scoring across all outputs
- **Parseltongue** — Input perturbation engine for red-teaming: 33 trigger techniques across 3 intensity tiers (leetspeak, braille, Morse, Unicode substitution, phonetic, bubble text)
- **AutoTune** — Context-adaptive sampling parameter engine with EMA-based online learning from thumbs up/down feedback
- **STM Modules** — Semantic Transformation Modules that normalize outputs in real-time (hedge reducer, direct mode, curiosity bias)
- **Single-file deployment** — No build step, no dependencies; runs from a single `index.html` on any static host
- **Privacy-first** — API key stays in the browser; lightweight telemetry is opt-out; no cookies, no PII

## Deployment

```bash
git clone https://github.com/elder-plinius/G0DM0D3.git
cd G0DM0D3
open index.html
```

Requires an [OpenRouter API key](https://openrouter.ai/keys). Also deployable to GitHub Pages, Vercel, Cloudflare Pages, or Netlify.
