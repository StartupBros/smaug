Autocontext is a recursive self-improving harness designed to help AI agents (and future iterations of those agents) succeed on any task. Originally built as a commercial tool, it was open-sourced due to community excitement around Karpathy's autoresearch approach.

The tool enables agents to autonomously improve their own performance by iterating on solutions and learning from failure patterns.

## Key Features

- **Recursive Self-Improvement**: Agents can improve their own performance over iterations
- **Failure Learning**: Systems learn from failures and adapt automatically
- **Iterative Optimization**: Continuous refinement of agent outputs and strategies
- **Multi-Agent Support**: Designed to work with multiple agent iterations

## Background

Inspired by Karpathy's autoresearch project, which demonstrated that agents can autonomously:
- Identify areas for improvement
- Generate candidate modifications
- Test and validate improvements
- Implement validated changes

Karpathy's autoresearch achieved approximately 20 improvements to validation loss on a 2-day optimization run, with all changes being additive improvements.
