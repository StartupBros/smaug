Google Workspace CLI is a command-line interface built for both humans and AI agents to interact with Google Workspace APIs. It provides access to Google Drive, Gmail, Calendar, and all other Workspace APIs with 40+ pre-built agent skills included.

The CLI makes it easy to automate workflows and integrate Workspace services programmatically, with agent-first design that enables autonomous operations.

## Key Features

- Access to all Google Workspace APIs (Drive, Gmail, Calendar, etc.)
- 40+ agent skills pre-built and ready to use
- Built for both human operators and autonomous agents
- CLI-first approach for easy integration
- Support for common automation workflows
