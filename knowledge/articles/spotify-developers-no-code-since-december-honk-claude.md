TechCrunch report on Spotify's internal AI coding system called "Honk," built on top of Claude Code, which has enabled the company's top engineers to stop writing manual code entirely since December 2025. The system shipped 50+ new features throughout 2025 and now allows engineers to deploy real-time bug fixes and new features directly from a phone during a commute. The piece highlights Spotify as a concrete production-scale example of AI dramatically accelerating engineering velocity at a major consumer tech company.

## Key Takeaways

- Spotify's "Honk" system is powered by Claude Code and has been in production since at least late 2025
- Top engineers have not hand-written code since December — all output is AI-generated and reviewed
- 50+ features shipped in 2025 alone, with the pace accelerating as the system matures
- Mobile-first deployment workflow: engineers can trigger fixes and deployments from a phone, removing the constraint of being at a workstation
- Represents a signal that AI-native engineering workflows are production reality at scale, not just developer productivity experiments
