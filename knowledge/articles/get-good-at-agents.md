A personal essay documenting a shift in work philosophy prompted by using Claude Code with Opus 4.5. The author's central realization: taking existing work habits into the agent era is fundamentally wrong. Prior habits — micromanaging, working on small tasks in isolation, grinding longer hours — actively limit the uplift agents can provide. The right direction is more open-ended, more ambitious, and more asynchronous task definition. The human's role shifts from executing with tools to directing an army.

The author draws an analogy: every engineer needs to learn system design; every researcher needs to run a lab; agents push humans up the org chart. The agent era requires cultivating a different mental state — more peace, more clarity at the directing level — rather than just skill at individual execution. Working more is no longer a lasting edge when multiple agents can execute in parallel.

Concrete workflow shared: GPT-5 Pro for planning, Claude Code with Opus 4.5 for implementation, with Claude Code passing detailed prompts back to GPT-5 Pro for deep research when stuck. Codex with GPT-5.2 on high thinking effort is noted as more meticulous but harder to extract best performance from. The author sees GPT Pro as "a strong agent trapped in the wrong UX."

## Key Takeaways

- Bringing pre-agent work habits into the agent era limits uplift — the change required is more fundamental than adopting new tools
- Human value shifts to ambitious, open-ended, asynchronous direction rather than hands-on execution of subtasks
- Cultivating focus and calm at the directing level becomes more important than maximizing individual output hours
- Practical stack: GPT-5 Pro planning + Claude Code (Opus 4.5) implementation + GPT-5 Pro deep research for difficult sub-problems
- Being early to agent workflows is an advantage, but hard work alone stops being a durable edge when parallel agents multiply throughput
- The shift feels larger than the earlier ChatGPT transition — this changes what you choose to work on, not just how you do chosen work
