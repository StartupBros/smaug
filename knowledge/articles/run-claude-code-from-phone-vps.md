A practical walkthrough for running Claude Code on a $4.09/month Hetzner VPS, accessible from any phone via the Terminus SSH terminal app. The guide covers provisioning the server, configuring SSH keys, installing Claude Code via the official install script, and fixing PATH issues. The core insight is that Claude Code only needs a persistent Linux environment — not a local laptop — so a cheap cloud server plus a mobile terminal app provides 24/7 access while commuting or traveling.

## Key Takeaways

- A Hetzner CX11 (~$4.09/month) is sufficient for running Claude Code
- Terminus (iOS/Android) provides a capable SSH terminal from a phone
- Total setup time is about 15 minutes; the environment persists between sessions
- The same VPS can host repos you clone and work on from anywhere
- Common pitfalls: forgetting to configure SSH keys on both ends, and skipping the PATH export step
