A step-by-step teardown of a high-converting quiz funnel (Kivo's Asian Walking program targeting menopausal women) that breaks down the psychological mechanism at every screen. The core thesis: quiz funnels work not because they collect data, but because each question is a micro-commitment device that moves the prospect from observer to invested participant. The author operates an analytics firm (Humblytics) and presents this as a replicable conversion framework applicable at any funnel stage.

The teardown covers ads through pricing page. Ads use conversational two-person dialogue that handles the primary objection ("I'm busy") upfront, with transformation specificity creating believability. The quiz itself sequences: identity segmentation (age bracket) → goal-based micro-commitment → belief reinforcement screens with no data collection → visual self-identification with neutral body images → temporal pain quantification → data collection disguised as BMI value delivery → personalized profile reveal → future-paced transformation with specific target date → pricing page with full trigger stack (countdown timer, daily cost reframe, social proof).

Key insight throughout: every screen is either building commitment, handling an objection, or delivering perceived value — never just collecting a form field.

## Key Takeaways

- Each quiz question is a micro-commitment, not a data point — commitment-and-consistency bias drives completion and purchase
- "Loading" screens are prime social proof real estate; use them to display testimonials and user counts
- BMI feedback screen is "data collection disguised as value delivery" — the gap it reveals is what your product fills
- Future-pacing with a specific date tied to a personal event ("the wedding") makes abstract goals emotionally real
- Testimonials at point of purchase should mirror objections verbatim ("I used to overcomplicate fitness")
- Per-day price reframing ($0.33/day vs. $9.99/month) frames cost relative to trivial alternatives
- "Forget 10,000 steps" — proactively demolish conventional wisdom objections before they surface
