A production scriptwriting system built on 20 specialized Claude Code agents that has generated over $10M in client revenue and 50M+ video views. The key insight is that single-agent "write the whole script" approaches produce garbage — specialization and gating are what make AI writing competitive with experienced human writers.

The pipeline has three phases. Research runs automated sweeps across YouTube (15 keyword searches across three time windows, ceiling/floor pattern analysis), Reddit (viral thread mining for customer pain quotes), and X (5,000 posts via API scored by engagement, with quote-tweet ratio as a proxy for controversy). Everything is indexed as ammunition for downstream agents.

The Writing Pipeline uses sequenced specialized agents: Hook Agent writes 4 data-backed hooks with 3 full iterations each, scored across 5 dimensions by a Hook Manager who gates nothing below 10/10. The same structure applies to body, CTA, and every other section. No agent ships without passing its own boss.

The Weapons Check scores every line independently on Invention Novelty (does this feel like a genuine breakthrough?) and Copy Intensity (does someone feel something, not just understand something?). Lines failing either dimension get rewritten; pure filler gets cut. Hard character budgets enforce runtime discipline throughout.

Output is a Google Doc with three tabs: Research, Working Script (full iteration/diagnosis trail), and Final Script (clean, 4 hook options, 2 CTA options). Humans do final tactical line-by-line editing from there.

## Key Takeaways

- Specialization beats single-agent: each agent has exactly one job and its own quality bar
- Gating is mandatory: nothing advances without clearing the boss agent's threshold
- Research phase is the foundation — hundreds of data points, not one search
- Weapons Check is the differentiator: novelty AND intensity both required, not either/or
- System averages 2M views per launch video; pricing starts at $100K per engagement
