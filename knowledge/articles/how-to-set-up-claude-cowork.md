## Summary

Comprehensive guide to Claude Cowork — Anthropic's desktop agentic product positioned as "Claude Code for everyone" (no terminal, no CLI). Published shortly after Cowork launched in January 2026.

## Five Core Features

1. **File System Access** — Claude reads/writes to a local folder on your machine. Eliminates the upload/download cycle. Key setup: create a "Claude Context" folder with `about-me.md`, `brand-voice.md`, `working-style.md`.

2. **AskUserQuestion** — Instead of guessing at ambiguous prompts, Cowork generates a structured clarification form. Use the prompt: *"DO NOT start working yet. First, ask me clarifying questions..."* The article's key insight: ChatGPT trained users to write better prompts; Cowork trains users to build better context files.

3. **Plugins** — Pre-built specialist packs for specific job functions. Anthropic shipped 11 official plugins at launch: Productivity, Marketing, Sales, Finance, Data Analysis, Legal, Product Management, Customer Support, Enterprise Search, Biology Research. Access via `+` button → Plugins, or `claude.com/plugins`. Use slash commands (`/`) after installing.

4. **Instructions (Global & Folder)** — Persistent memory that loads automatically each session. Set in Settings → Cowork → Global Instructions. Folder-level instructions enable per-client context loading.

5. **Connectors** — Live integrations with Slack, Google Drive, Notion, Figma, Asana, 50+ tools. Free on all plans. Configured once in Settings → Connectors.

## Pricing

- Pro: $20/month
- Max: $100–$200/month (for heavier usage)

## Limitations Noted

- No cross-session memory (workaround: context files + global instructions)
- Tasks stop if the desktop app is closed (sleep the computer instead)
- Higher token usage than standard chat for complex tasks
- Desktop only — no mobile, no browser Cowork, no cross-device sync
- No image generation
- Still a "research preview" — agent safety under active development

## 30-Minute Setup Checklist

1. Download desktop app at `claude.com/download`
2. Create three context files (`about-me.md`, `brand-voice.md`, `working-style.md`) in a "Claude Context" folder
3. Set global instructions in Settings → Cowork
4. Run first Cowork task against the context folder
5. Install one role-relevant plugin
6. Connect one live tool (Slack, Drive, or Notion)
