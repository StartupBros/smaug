A step-by-step setup guide aimed at non-technical marketers who want to use Claude Code to automate marketing workflows without writing code. The author — self-described as "as non-technical as a potato" — walks through installing VS Code, installing Claude Code via curl, adding the official skills pack (12 skills including PPTX generation), and launching through the VS Code extension for a chat-style interface. The guide then pivots to the real value: turning personal marketing knowledge into reusable Claude Code skills.

The core insight is that skills act as codified SOPs — you can turn any methodology (Russell Brunson landing page frameworks, Alex Hormozi offer creation, James Clear productivity systems, or online courses you've purchased) into a skill file, and Claude will apply it automatically. The author uses this to run scrapers analyzing competitor landing pages, write copy using direct response frameworks, and generate product ideas autonomously while away from the keyboard.

## Key Takeaways

- Skills encode personal IP and frameworks so Claude applies your methodology automatically, not just generically
- Non-technical users can bootstrap from zero with the Anthropic skills pack (12 free skills at skills.sh)
- The VS Code Claude extension provides a cleaner chat UI than the raw terminal interface
- Multiple Claude Code agents can run simultaneously by opening parallel chats
- Treat Claude Code setup as a 15-minute investment that compounds indefinitely, not a bookmark to revisit
